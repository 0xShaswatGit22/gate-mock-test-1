# GATE 2026 Mock Test Simulator 🚀

A beautiful, interactive **GATE 2026** online mock test platform built with HTML, CSS & Vanilla JavaScript.  
Perfect for students preparing for GATE — complete with timer, question palette, marking for review, login/signup, and submission!

## ✨ Features

- 🖥️ Clean & modern UI with Poppins font
- 🔐 Login + Sign Up system (localStorage based)
- ⏱️ 3-hour countdown timer
- 📊 Question palette with 4 status colors:
  - Not Visited ⚪
  - Not Answered 🟡
  - Answered 🟢
  - Marked for Review 🔵
- ✅ Save & Next / Mark & Next / Clear Response buttons
- 📝 Real-time question navigation
- ✅ Final Submit button with score calculation
- 🐞 Debug panel to view registered users & results
- 📱 Responsive design (mobile + desktop friendly)

## 🛠️ Tech Stack

- HTML5
- CSS3 (with Flexbox & modern styling)
- Vanilla JavaScript (no frameworks)
- Google Fonts (Poppins)
- localStorage for users, questions & responses
